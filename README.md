# skopec.dev

The source code for my own website. This repository contains the HTML, CSS, and JS files for my portfolio and link tree, [skopec.dev](https://skopec.dev).

## About

This repo holds the main source files of the site — the core HTML pages, stylesheet, and script. Besides that, you'll also find:

- `sitemap.xml` – sitemap for search engines
- `robots.txt` – crawler rules
- `favicon.svg` and the `images/` folder – icons and photos
- `.idea/` – IDE config files
